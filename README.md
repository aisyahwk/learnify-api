Learnify (Web App)

Learnify adalah website pembelajaran yang membantu siswa memahami materi secara lebih mudah melalui penyajian yang terstruktur dan jelas. Website ini menampilkan materi, submateri, serta penjelasan lengkap beserta contoh kode (khusus materi IT seperti HTML, CSS, JavaScript).

Kegunaan Aplikasi Learnify : 
-Membantu siswa belajar mandiri
-Menyediakan materi yang tersusun
-Menampilkan penjelasan + contoh kode
-Mempermudah pemahaman konsep dasar programming

Cara pemakaian :
-Buka website learnify di browser
-Pilih materi (HTML, CSS, JavaScript)
-Masuk ke submateri, ada 8 submateri di tiap materi nya 
-Baca penjelasan, dan contoh penggunaan kode

Fitur Utama :
-List materi dan submateri pembelajaran
-Konten penjelasan lengkap
-Contoh kode 
-Terhubung dengan API (data realtime)

Manfaat :
-Cocok buat pemula
-Bisa diakses kapan aja
